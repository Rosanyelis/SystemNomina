# SYSTEM_DESIGN.md

# Oficina Contable Mendoza Bellorin (OCMB)

Versión: 1.0

---

# OBJETIVO

Este documento define las reglas visuales, de experiencia de usuario y comportamiento de interfaz que deben respetarse en cualquier producto digital perteneciente a Oficina Contable Mendoza Bellorin.

Este documento tiene prioridad sobre cualquier decisión visual generada automáticamente por herramientas de IA.

---

# IDENTIDAD DE MARCA

La marca debe transmitir:

* Confianza
* Seguridad
* Precisión
* Profesionalismo
* Transparencia
* Orden

Toda interfaz debe percibirse como:

* Corporativa
* Elegante
* Moderna
* Clara
* Estable

Nunca debe percibirse como:

* Infantil
* Informal
* Experimental
* Tecnológica extrema
* Startup disruptiva

* Shake
* Flash
